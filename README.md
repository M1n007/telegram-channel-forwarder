# Telegram Channel Forwarder

Cara menjalankan :

```python
$ pip3 install -r requirements.txt
$ python3 index.py
```


Cara mendapatkan API ID dan API Hash

1. Login Ke [telegram account](https://my.telegram.org/)
2. Lalu klik "API development tools" and isi detail aplikasimu (only app title and short name required)
3. Terakhir, klik "Create application"
